# Playbook: Investigation Brief

## Purpose
When we don't have enough information to act, produce a focused investigation brief
that outlines what we need to learn and how to learn it.

## Structure

### 1. What We Know
Summarize the pattern and existing evidence. Cite problem IDs.

### 2. What We Don't Know
Specific questions that need answers before we can propose a solution.

### 3. Investigation Plan
For each question:
- **Question**: What do we need to find out?
- **Method**: How will we answer it? (interview, data pull, experiment, observation)
- **Who**: Who can provide this information?
- **Timeline**: How long should this take?

### 4. Decision Point
What will we do with the answers? Define:
- If we learn X → proceed with [type of solution]
- If we learn Y → reconsider the pattern

### 5. Deadline
When should this investigation complete? Don't let it drag.

## Guidelines
- Maximum 5 open questions. If more, the problem isn't defined well enough.
- Every question must have a concrete method — no "we should look into it"
- Set a hard deadline. Investigations without deadlines become procrastination.
