# Playbook: Action Plan

## Purpose
Create a step-by-step implementation plan that a team can execute.

## Structure

### 1. Objective (1 sentence)
What are we trying to achieve?

### 2. Background
Which pattern and hypothesis does this address? (cite IDs)

### 3. Prerequisites
What must be true before we start?

### 4. Steps
Numbered list. Each step includes:
- **Action**: What to do
- **Owner**: Which role is responsible
- **Duration**: Estimated time
- **Dependencies**: Which steps must complete first
- **Done when**: How to verify this step is complete

### 5. Success Criteria
How do we know the whole plan worked? Reference the hypothesis test_criteria.

### 6. Rollback Plan
If things go wrong, how do we undo?

### 7. Timeline Summary
A simple Gantt-style list: Step 1 (Week 1), Step 2 (Week 1-2), etc.

## Guidelines
- Maximum 10 steps. If more are needed, break into phases.
- Every step must have a clear "done when" — no ambiguous completion.
- Prefer parallel steps where possible to reduce total time.
